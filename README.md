# zdrav-bot

Бот, проверяющий расписание (раз в 10 минут) и выводящий список доступных талонов с https://zdrav.mosreg.ru

## Как пользоваться

1. Откройте https://t.me/zdrav_mosreg_subscribe_bot
2. Введите `/start`
3. Следуйте инструкциям

## Самостоятельное развёртывание

- создайте своего telegram бота
- создайте свой firebase проект с firestore database
- создайте `.env` файл, укажите значения (пример значений в `.env.example`)
- `npm run build && npm run start`

## Отказ от ответственности

- Данный бот не является официальным и существует лишь для того, чтобы находить свободные талоны на https://zdrav.mosreg.ru
- Данный бот никак не создаёт/удаляет/редактирует записи ко врачу, все операции работают на чтение API, никаких изменений не просходит
- Все введёные данные хранятся в https://firebase.google.com и не передаются третьим лицам
